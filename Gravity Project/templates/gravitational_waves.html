Gravitational Wave Analysis

Machine Learning for Black Hole-Neutron Star Merges

Project Overview

I analyzed gravitational wave signals from black hole-neutron star merging events using machine learning. When these objects begin to merge together, they create grsvity waves that are polarized. These polarizations are called h+ and hx. Here, this model predicts rhx

Dataset

For this project, I used 5 BHNS simulations provided by the NASA database. Each simulation consisted of three columns. Time(s), rh+(cm), and rhx(cm). "r" is simply the distance from the observer. For a more accurate description, I am using time and rh+ to predict the cross-polarization gravitational wave strain amplitude multiplied by distance.

ML Approach

I trained Linear Regression, Lasso Regression and Random Forest models to predict rhx amplitude from time and rh+ values. Random Forest achieved the best performance.

Methodology

Data Prep
  • Loaded 5 SPH files from gz file (Simulation data)
  • Combined datasets(all diffrent scenarios)
  • 70/30 train/test split
Training
  • Linear and Lasso Regression baseline
  • Random Forest (100 trees)
  • Cross-validation(3)
Evaluation
  • RMSE calculations
  • R^2 scores
  • Best model selection

Results

While Linear and Lasso Regression both performed weakly with nearly identical RMSEs of 31735.3 and 31735.2 respectivly, the also had abysmal R^2 values of ~ -6*10^-05. Random Forest managed to successfully capture a non-linear relationship in the GW data and yielded a RMSE of 5,881.3 and a R^2 of 0.965. this proves that machine learning can be a useful tool in capturing real physics via simulation data, but must must be applied with the correct approch.
